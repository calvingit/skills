
## 如何安装？

**安装到 AI 助手**

```bash
# 默认：先安装个人技能包，再安装 index.json 全部条目
python3 ./scripts/install.py

# 跳过默认技能包，只安装 index.json 条目
python3 ./scripts/install.py --skip-default

# 指定 Agent（可重复或逗号分隔）
python3 ./scripts/install.py -a claude-code -a codex

# 全局安装
python3 ./scripts/install.py -g
```

每个技能调用 `npx skills add <url>`，默认附带 `-y`（无需确认）。
安装完成后自动执行 `post_install.py` 检查必要 CLI 工具是否就绪（ctx7、tvly、agent-browser）。

**删除技能或分类**

```bash
# 删除单个技能（调用 npx skills remove）
python3 ./scripts/uninstall.py --skill chrome-devtools

# 删除整个分类（展开为该分类下所有技能逐一 remove）
python3 ./scripts/uninstall.py --category Design

# 预览卸载命令（不实际执行）
python3 ./scripts/uninstall.py --category Tool --dry-run
```
