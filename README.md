## 脚本说明

**安装（scripts/install.py）**

```bash
# 默认：交互式选择分类（内置 Skills 默认勾选），并选择 Agent 客户端
python3 ./scripts/install.py

# 非交互安装：安装全部外部分类 + 内置 Skills
python3 ./scripts/install.py --no-interactive

# 仅安装指定外部分类（不包含内置 Skills）
python3 ./scripts/install.py --category Design

# 指定安装目标目录
python3 ./scripts/install.py --target ~/.agents/skills
```

安装脚本会把技能复制到目标目录，并为 Claude/Codex/Hermes 创建对应软链接。
若 `external-skills/` 为空，会自动触发 `scripts/download.py` 先下载外部技能。

**卸载（scripts/uninstall.py）**

```bash
# 卸载单个技能
python3 ./scripts/uninstall.py --skill chrome-devtools

# 卸载整个分类
python3 ./scripts/uninstall.py --category Design

# 卸载全部已安装内容
python3 ./scripts/uninstall.py --all

# 预览卸载操作（不实际执行）
python3 ./scripts/uninstall.py --category Tool --dry-run
```

**下载外部技能（scripts/download.py）**

```bash
# 下载全部外部技能到 external-skills/
python3 ./scripts/download.py

# 只下载指定分类
python3 ./scripts/download.py --category Tool

# 清理缓存后重新下载
python3 ./scripts/download.py --clean-cache
```

**README 与一致性校验**

```bash
# 根据 index.json 重新渲染 README 的技能列表区块
python3 ./scripts/render_readme_skills.py

# 校验 index.json、README、external-skills 是否一致
python3 ./scripts/validate_consistency.py

# 以 JSON 输出校验结果
python3 ./scripts/validate_consistency.py --json
```

**安装后依赖检查（scripts/post_install.py）**

```bash
# 检查当前索引中涉及的 CLI 依赖与环境变量
python3 ./scripts/post_install.py

# 仅检查指定 skill 的依赖（支持逗号分隔）
python3 ./scripts/post_install.py --installed-skills context7-cli,tavily-search
```
